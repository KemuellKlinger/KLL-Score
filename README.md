🏁 KLL Championship Score

Sistema web simples para controle de pontuação de equipes inspirado no formato da FLL (FIRST LEGO League). Ideal para competições internas, treinamentos ou oficinas de robótica.

🎯Objetivo

Este projeto foi desenvolvido para facilitar a avaliação de equipes em desafios de robótica educacional, tornando o processo mais rápido, visual e organizado.
